Selamat datang di repositori portofolio Excel saya. Saya mendedikasikan tempat ini untuk mendokumentasikan perjalanan saya dalam mempelajari dan menguasai analisis data menggunakan Microsoft Excel dan Google Sheets.

Di sini, Anda akan menemukan berbagai proyek latihan yang menunjukkan kemampuan saya dalam mengelola, membersihkan, dan menganalisis data.

Saya memiliki kemampuan dan pemahaman yang kuat dalam menggunakan fungsi-fungsi inti untuk analisis data, termasuk:

Pencarian Data Lanjutan: Menggunakan VLOOKUP dan XLOOKUP untuk menemukan dan menghubungkan data di berbagai tabel, termasuk mencari data ke kiri (XLOOKUP).

Logika Bersyarat: Menerapkan fungsi IF dan IFERROR untuk membuat keputusan otomatis dan menangani error dengan rapi.

Analisis Data Bersyarat: Menggunakan SUMIF dan SUMIFS untuk menghitung total berdasarkan satu atau beberapa kriteria.

Manajemen Data: Menguasai Sort(Pengurutan) dan Filter (Penyaringan) untuk merapikan data set yang besar.

Validasi Data: Membuat menu dropdown untuk memastikan input data yang bersih dan bebas kesalahan.

Formula Dasar: Menggunakan SUM, COUNTA, dan perhitungan aritmatika dasar.
